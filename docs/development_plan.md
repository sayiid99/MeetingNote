# Development Plan

## Phase 1 Build Order

1. Application shell, logging, SQLite initialization
2. File import and ffmpeg preprocessing
3. ASR provider abstraction and FunASR implementation
4. Transcript result view, editing, and audio playback sync
5. GGUF model management and LLM provider
6. Full-document Chinese-English translation service
7. Meeting summary service
8. History records and exports
9. Packaging and Windows 11 smoke tests

## Phase 1 Non-Goals

Self-media writing, voiceover scripts, platform-specific articles, and text-to-video scripts are intentionally excluded from Phase 1.
